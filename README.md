# 中国大陆居民投资美股指南

本仓库是一份面向中国大陆居民的美股开户、账户选择、入金、出金与合规风险手册。目标不是鼓励任何人绕开规则，而是把常见路径的材料要求、费用、稳定性、信息交换影响和实际操作难度放在同一张桌面上，方便读者做更清醒的判断。

> 免责声明：本文只做信息整理，不构成投资、税务、法律、外汇或移民建议。跨境金融政策变化很快，动手前请以券商、银行、监管机构和专业顾问的最新口径为准。

## 一句话路线图

| 目标 | 推荐优先级 | 典型组合 | 核心代价 |
| --- | --- | --- | --- |
| 完全省心、少碰跨境细节 | 高 | 境内 QDII / 互联互通类产品 | 不能自由买个股，可能有溢价 |
| 普通人直接买美股 | 高 | 香港/美国银行 + 美国券商 + 主动核验纳税义务 | 外汇用途、税务申报、银行风控都要自己负责 |
| 更强的美元系统能力 | 中 | 美国银行 + 美国券商 + ITIN 或赴美线下开户 | 成本高、周期长、资料要求多 |
| 资金长期留在海外 | 中 | 美国券商 + 美国银行/消费场景 | 要提前考虑遗产税、继承和长期合规 |
| 加密或非传统通道 | 低 | 加密平台/欧洲金融科技账户 + 券商 | 只适合熟悉加密和风控的人，政策不确定性高 |

## 新手优先做什么

1. 先确定自己属于哪类风险偏好：只想合规省心、愿意主动报税、还是能承受更复杂的跨境账户管理。
2. 再选券商。普通大陆居民常见选择是 Schwab International、Firstrade、Interactive Brokers、BBAE/BIT 等，但各家开户口径会变化。
3. 同步准备银行账户。香港账户容易上手，美国账户更接近美元体系核心，但门槛更高。
4. 第一次入金只做小额测试。确认姓名、地址、附言、入金通知、到账路径都正确，再扩大金额。
5. 保存所有材料。开户资料、W-8BEN、银行流水、券商月结单、交易记录和汇款凭证都应长期归档。

## 常见账户组合

| 组合 | 适合人群 | 优点 | 风险与缺点 |
| --- | --- | --- | --- |
| 香港银行 + 美国券商 | 有港卡、想尽快上手 | 开户相对容易，链路短，资料少 | 香港账户参与 CRS，年底余额和相关收入可能被交换 |
| 美国银行 + 美国券商 | 有美签/ITIN/美国地址资源 | ACH 方便，美元体系内转账顺滑 | 开户门槛高，远程账户风控较严 |
| 香港银行 + Wise/中转账户 + 美国券商 | 想降低汇款成本的人 | 换汇和 ACH 体验好，费用可能较低 | 中转机构风控严格，也可能参与信息交换 |
| 汇丰中港美 + 美国券商 | 汇丰卓越/优先理财用户 | 同集团跨境转账体验好 | 资产门槛高，地区口径差异大 |
| 美国券商 + 境内 QDII 备选 | 想兼顾直接投资和合规备胎 | 可以分散路径依赖 | 需要同时管理境内外资产和记录 |
