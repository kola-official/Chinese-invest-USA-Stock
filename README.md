> 免责声明：本文只做信息整理，不构成投资、税务、法律、外汇或移民建议。跨境金融规则变化很快，开户、汇款、申报前请以官方口径和专业意见为准。

## 先选路线

| 需求 | 可考虑的方式 | 主要成本 |
| --- | --- | --- |
| 只想省心配置美股指数 | 境内 QDII、ETF、互联互通类产品 | 溢价、额度、无法自由选择个股 |
| 想直接买美股个股/ETF | 所选券商 + 香港/美国/中转账户 | 开户资料、汇款审核、税务资料留存 |
| 想提高美元体系可用性 | 美国银行能力 + ITIN 或赴美线下开户 | 时间、费用、地址和身份材料 |
| 已熟悉加密资产 | 加密平台或金融科技账户作为补充通道 | 平台风控、政策变化、资金来源解释 |

## 新手执行顺序

1. 先确定自己是否接受直接境外证券投资带来的税务、外汇和银行风控责任。
2. 再选券商：优先看开户门槛、监管实体、出入金方式和长期稳定性。
3. 反推资金账户：已有港卡就先小额试转；有美国银行能力则优先研究 ACH；没有两者再考虑 Wise 等中转账户。
4. 首笔只做小额。确认姓名、附言、入金通知、到账路径全部正确，再扩大金额。
5. 保存所有记录：开户材料、W-8BEN、银行回单、券商月结单、交易流水、年度税表。
