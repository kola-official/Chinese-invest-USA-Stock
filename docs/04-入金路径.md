# 入金路径

入金的关键是“同名、可解释、小额测试”。同名转账能显著降低银行和券商风控难度；可解释意味着每一步都有回单和资金来源；小额测试能在大额前暴露账号、附言、币种和到账规则的问题。

## 入金方案总览

| 方案 | 链路 | 成本 | 速度 | 稳定性 | 适合人群 | 主要风险 |
| --- | --- | --- | --- | --- | --- | --- |
| A | 内地银行 -> 香港银行 -> 美国券商 | 中 | 1-3 工作日 | 高 | 已有港卡的新手 | 香港账户参与 CRS，电汇费用 |
| B | 内地银行 -> 香港银行 -> Wise/中转 -> 美国券商 | 低到中 | 1-3 工作日 | 中 | 想降低换汇和转账成本的人 | 中转账户风控，信息交换 |
| C | 内地银行 -> 香港银行 -> 美国银行 -> 美国券商 | 中 | 2-5 工作日 | 高 | 已有美卡或汇丰中港美账户 | 步骤多，银行审核 |
| D | 内地银行 -> 美国银行 -> 美国券商 | 中 | 1-3 工作日 | 中到高 | 有稳定美国银行账户的人 | 境内银行可能拒绝汇款用途 |
| E | 内地银行 -> Wise/中转 -> 美国券商 | 中 | 1-3 工作日 | 低到中 | 无港卡/美卡、只做小额测试 | 成功率不稳定，冻结风险 |
| F | 加密/欧洲金融科技账户 -> 券商 | 不确定 | 1-3 工作日 | 低到中 | 熟悉加密且能承担高风险的人 | 平台、政策、KYC、链上资金来源 |

## 方案 A：港卡直入美国券商

适合第一次跑通路径。步骤通常是：

1. 在券商端获取电汇收款信息，确认是否需要先创建入金通知。
2. 在香港银行端发起同名汇款，币种按券商要求选择美元或港币。
3. 按券商要求填写收款人、收款银行、账户号、SWIFT、附言或 FFC 信息。
4. 到账后下载银行回单和券商入金记录。

优点是链路直观、材料容易解释。缺点是电汇费和汇率点差可能不低，且香港账户属于 CRS 地区。

## 方案 B：港卡 + 中转账户

适合希望降低手续费、使用 ACH 或改善换汇体验的人。常见逻辑是：

```mermaid
flowchart LR
  A["内地银行购汇"] --> B["香港银行"]
  B --> C["Wise 或其他中转账户"]
  C --> D["美国券商"]
```

这类方案的收益在于转账费用和到账速度，但中转账户并非传统银行，风控可能更敏感。建议只做同名账户之间的资金流动，避免多人收付款、频繁拆分和无法解释的资金来源。

## 方案 C：港卡 + 美卡 + 券商

如果已经有美国银行账户，最顺滑的长期路径往往是先把资金放入美国银行，再通过 ACH 入金券商。

```mermaid
flowchart LR
  A["内地银行"] --> B["香港银行"]
  B --> C["美国银行"]
  C --> D["美国券商 ACH"]
```

这种路径的优势是美元体系内部转账体验好，缺点是美国银行账户本身不容易拿到，远程开户账户也可能因非典型资金流动触发风控。

## 方案 D：内地直汇美国银行

理论上链路最短，但现实中境内银行会审核购汇用途和收款原因。直接把资金从内地汇到美国银行或券商，可能被要求补充材料、修改用途，甚至直接退回。若要尝试，务必先小额测试，并确认用途表述真实合规。

## 方案 E：无港卡/美卡时的临时路径

无港卡、无美卡时，有人会尝试通过 Wise 或其他中转账户直接入金券商。这条路的可复制性较差，成功与否受银行、币种、账户资料、时间窗口影响很大。不要把它当成大额资金的主路径。

## 方案 F：加密或欧洲金融科技通道

这类路径只适合已经熟悉加密资产、稳定币、链上转账、CEX/KYC 和金融科技账户风控的人。普通投资者不应为了买美股从零学习加密入金。

主要注意点：

- 平台是否仍接受中国大陆居民。
- 平台实体在哪个司法辖区，是否参与 CRS 或 CARF。
- 稳定币来源是否可解释。
- 是否能以同名账户向券商出款。
- 发生冻结或退汇时，客服和申诉路径是否清楚。

## 嘉信入金特别检查

| 项目 | 检查点 |
| --- | --- |
| 账户类型 | Schwab International 还是其他账户 |
| 入金币种 | 美元、港币或其他币种是否会被自动换汇 |
| 收款信息 | 是否需要 FFC/For Further Credit 或附言写入个人账户号 |
| ACH | 是否使用美国银行或支持 ACH 的中转账户 |
| 首次入金 | 是否有金额、时间或账户激活限制 |
| 出金绑定 | 入金账户是否会影响未来出金账户审核 |

## 入金前检查清单

| 检查项 | 说明 |
| --- | --- |
| 姓名一致 | 拼音、顺序、空格尽量与券商一致 |
| 地址一致 | 银行和券商地址差异过大可能触发补材料 |
| 附言准确 | 券商公户入金时尤其重要 |
| 小额测试 | 首笔不要大额 |
| 回单保存 | 银行汇出、券商入账、换汇记录都保存 |
| 时间避让 | CRS 地区账户避免长期年末大额闲置，具体申报义务另行核验 |
