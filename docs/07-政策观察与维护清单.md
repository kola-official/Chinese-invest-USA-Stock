# 政策观察与维护清单

跨境投资路径会随政策、银行风控和平台开户口径变化。本清单用于记录长期维护时应检查的事项。

## 重点观察项

| 类别 | 看什么 | 为什么 |
| --- | --- | --- |
| 关注平台 | BIT、Wise 是否仍接受中国大陆用户 | 决定相关链接和操作说明是否仍可使用 |
| 入金方式 | 电汇、ACH、Wise、本地转账、稳定币等是否可用 | 决定资金能否进入账户 |
| 出金方式 | 出金到香港银行、美国银行、Wise 或其他账户是否稳定 | 决定未来退出路径 |
| CRS | 参与地区、报告字段、年度交换口径是否变化 | 影响境外账户摘要数据如何回流 |
| CARF | 加密平台报告义务和中国参与情况 | 影响加密通道的长期确定性 |
| 外汇规则 | 个人购汇用途、境外投资申报、资金回流要求 | 影响整体合规风险 |
| 美国税务 | W-8BEN、股息预扣、遗产税、ITIN 规则 | 影响长期持有成本 |

## 维护规则

- 只收录可核验的链接和渠道信息。
- BIT 和 Wise 之外的银行和券商只写账户类型。
- 不把单个客服答复或个案当成稳定规则，除非有多个来源印证。
- 政策类信息优先引用监管机构、银行、券商或税务机构公告。
- 高风险路径必须写清楚适用人群和失败后果。

## 年度自查

| 频率 | 自查内容 |
| --- | --- |
| 每次开户前 | 链接是否可核验、材料是否变化、是否仍接受所在地区 |
| 每次入金前 | 收款信息、附言、币种、手续费、到账时间是否变化 |
| 每季度 | 手机号、邮箱、地址、登录设备是否仍可用 |
| 每年年末 | CRS 地区账户余额、股息和税务资料是否整理好 |
| 每年报税季 | 年度税表、股息预扣、交易盈亏和申报义务 |
| 资产规模明显增加时 | 遗产税、继承安排、授权联系人和家人知情程度 |
