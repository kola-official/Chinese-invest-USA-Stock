# 政策观察与维护清单

跨境投资路径的最大变量不是某个 App 好不好用，而是政策、银行风控和券商开户口径不断变化。本章用于记录需要长期观察的项目，也方便后续维护者更新。

## 重点观察项

| 类别 | 观察内容 | 为什么重要 |
| --- | --- | --- |
| 券商开户 | 是否继续接受中国大陆居民，是否新增地址/签证/资金来源要求 | 直接决定新用户能否开户 |
| 银行开户 | 香港、美国、新加坡、欧洲账户是否收紧 | 决定入金和出金路径 |
| 入金方式 | 电汇、ACH、Wise、SEPA、稳定币等是否仍可用 | 决定资金能否进入券商 |
| 出金方式 | 券商出金到港卡、美卡、中转账户是否稳定 | 决定未来退出路径 |
| CRS | 报告范围、参与地区、数据颗粒度是否变化 | 影响税务信息可见性 |
| CARF | 加密平台报告义务和中国参与情况 | 影响加密通道可持续性 |
| 外汇规则 | 个人购汇用途、境外投资申报、资金回流要求 | 影响整体合规性 |
| 美国税务 | W-8BEN、股息预扣、遗产税、ITIN 规则 | 影响长期持有成本 |

## 维护规则

- 任何开户、邀请、返佣、推广链接都统一保留为 `tobefill`，不放个人推广码。
- 只记录可被复核的信息，避免把单个网友的个案写成普遍规则。
- 涉及政策变化时，优先引用监管机构、银行、券商、税务机构公告。
- 如果只来自客服口径或社群反馈，应明确标注“待核验”。
- 高风险路径必须标明适用人群和失败后果。

## 读者自查清单

| 频率 | 自查内容 |
| --- | --- |
| 每次开户前 | 官网是否仍接受所在地区、材料是否变化、入口是否为官方入口 |
| 每次入金前 | 收款信息、附言、币种、手续费、到账时间是否变化 |
| 每季度 | 银行和券商是否要求补材料，手机号/邮箱/地址是否仍可用 |
| 每年年末 | CRS 地区账户余额、股息、税务申报资料是否整理完成 |
| 每年报税季 | 券商年度税表、股息预扣、交易盈亏和申报义务 |
| 资产规模明显增加时 | 遗产税、保险、信托、家人授权和继承安排 |
