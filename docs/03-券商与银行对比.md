# 券商与银行对比

本章用表格比较常见券商、银行和中转账户。信息会变化，表格只适合作为调研起点，不应替代开户前的官方核验。

## 美国券商对比

| 维度 | Schwab International | Firstrade | Interactive Brokers | BBAE / BIT 等其他平台 |
| --- | --- | --- | --- | --- |
| 定位 | 美国大型综合券商国际账户 | 面向华人用户较友好的美国券商 | 专业化、全球化、多市场券商 | 小型、特色或加密/RWA 相关平台 |
| 大陆居民开户 | 护照与地址资料，口径可能变化 | 护照为主，门槛相对低 | 可能要求非大陆地址证明 | 部分平台身份证或护照即可，需核验实体和监管 |
| 中文支持 | 有中文客服，但等待时间可能较长 | 中文支持较强 | 有中文支持，体验偏专业 | 取决于平台 |
| 产品范围 | 美股、ETF、期权等 | 美股、ETF、期权等 | 美股、港股、欧股、日股等多市场 | 通常以美股或特定产品为主 |
| 美股/ETF 佣金 | 常见零佣金 | 常见零佣金 | 阶梯或固定费率 | 需逐项核验 |
| 期权费用 | 通常按合约收费 | 费用较友好 | 按合约和方案收费 | 需逐项核验 |
| 入金能力 | 电汇、ACH、部分第三方路径 | 电汇、ACH、转仓等 | 电汇、ACH、Wise、SEPA、多币种等 | 取决于平台，部分支持稳定币 |
| 操作难度 | 中 | 低 | 中到高 | 差异很大 |
| 更适合 | 重视平台规模和长期稳定性 | 想快速上手、中文服务优先 | 多市场、多币种、专业工具需求 | 熟悉特定生态并能评估平台风险 |
| 开户入口 | tobefill | tobefill | tobefill | tobefill |

## 券商核验清单

| 要查什么 | 为什么重要 |
| --- | --- |
| 公司注册实体 | 判断是否美国实体、香港实体或其他司法辖区实体 |
| SEC / FINRA 登记 | 确认是否受美国证券监管 |
| SIPC 会员资格 | 判断客户资产保护框架 |
| 清算与托管安排 | 小券商尤其要看客户资产放在哪里 |
| 出入金限制 | 很多问题不是买不了股票，而是钱进不去或出不来 |
| 账户关闭规则 | 被要求转仓、清退或补材料时，资金是否能顺利处理 |

## 如何选择银行

| 情况 | 优先考虑 |
| --- | --- |
| 已有港卡 | 先用港卡跑小额入金，积累完整记录 |
| 有美签且能赴美 | 到美国线下开主流银行账户，体验通常最好 |
| 有汇丰卓越/优先理财 | 研究中港美同集团账户联动 |
| 无港卡无美卡 | 可以研究 Wise 或其他中转账户，但要接受成功率和风控不确定性 |
| 资金规模较大 | 先咨询税务、法律和私人银行，不要只按网上教程操作 |

## 费用关注点

| 费用 | 常见发生位置 |
| --- | --- |
| 境内购汇点差 | 人民币换港币/美元/其他币种 |
| 跨境电汇手续费 | 汇出行、中转行、收款行都可能收费 |
| 中转账户换汇费 | Wise 或其他金融科技账户 |
| 券商入金/退汇处理费 | 特别是信息填写错误时 |
| 账户月费 | 香港银行、美国银行、私行账户 |
| 平台费/交易费 | 小券商或特定平台可能收取 |
