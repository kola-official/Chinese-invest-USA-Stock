# 开户与账户组合

直接投资美股通常要拼出三个零件：交易账户、资金账户、必要时再加一个中转账户。很多失败案例不是券商没开成，而是钱卡在银行、中转机构或出金环节。

## 三个零件

| 账户 | 负责什么 | 选择时看什么 |
| --- | --- | --- |
| 券商账户 | 下单、持仓、出具月结单和税表 | 监管实体、开户资料、入金方式、中文支持 |
| 银行账户 | 购汇后汇出、接收券商出金、日常美元使用 | 所在司法辖区、同名转账、月费、风控口径 |
| 中转账户 | 换汇、低成本转账、桥接 ACH 或本地转账网络 | 是否参与信息交换、冻结后如何申诉、是否适合大额 |

判断一个组合能不能长期使用，不只看“能否开户”，还要看“钱能不能低摩擦地进去、出来，并且每一步都能解释”。

## 资金账户类型

| 类型 | 价值 | 短板 |
| --- | --- | --- |
| 香港银行账户 | 距离近、开户材料熟悉、可作为常见跨境汇款节点 | 香港参与 CRS，开户和汇款审核持续收紧 |
| 美国银行账户 | ACH 便利，美元体系内转账体验最好 | 远程开户难度较高，线下开户通常需要赴美、地址和更多材料 |
| 跨地区银行体系 | 多地账户联动顺滑，适合已有高等级客户 | 资产门槛高，不同地区客户经理口径差异大 |
| Wise 等中转账户 | 换汇和转账体验好，能补足部分 ACH/本地转账能力 | 风控敏感，不适合长期沉淀大额资金 |

## 常见组合

| 组合 | 适合场景 | 先做什么 |
| --- | --- | --- |
| 港卡 + 所选券商 | 已有香港账户，想尽快小额试转 | 用同名账户跑一笔小额电汇 |
| 美卡 + 所选券商 | 已有美国银行能力，重视长期美元闭环 | 先确认券商是否支持 ACH 绑定 |
| 港卡 + Wise + 所选券商 | 想降低换汇或转账成本 | 先确认 Wise 与券商之间的收付款规则 |
| 美国银行 + Wise + 所选券商 | 需要多币种或备用通道 | 明确哪个账户只做中转，哪个账户长期使用 |

| 材料 | 用途 |

## 小额试运行

1. 先在券商里创建入金通知或绑定银行账户。
2. 从本人同名账户发起小额转账。
3. 核对收款人、账号、银行代码、附言和币种。
4. 到账后下载银行回单和券商入账记录。
5. 确认月结单、税表和交易记录能正常导出。
