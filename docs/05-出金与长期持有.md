# 出金与长期持有

入金解决“怎么买”，出金解决“以后怎么解释、怎么花、怎么继承”。越早设计出口，后面越少临时补材料。

## 出口总览

| 出口 | 资金去向 | 适合场景 | 主要问题 |
| --- | --- | --- | --- |
| 继续留在美元资产里 | 券商、美元基金、美国银行 | 长期配置，不急着回款 | 遗产税、继承、账户维护 |
| 回到香港银行 | 券商 -> 香港银行 | 已准备税务资料，想回到常见跨境银行节点 | CRS 年度摘要、银行问询 |
| 进入美国银行后消费 | 券商 -> 美国银行 -> 消费 | 留学、旅行、美元消费、外卡支付 | 需要稳定美国银行账户 |
| Wise 小额承接 | 券商 -> Wise -> 境内或其他账户 | 小额、偶发、资料清晰 | 风控敏感，不适合大额长期沉淀 |
| 加密或特殊平台 | 券商/平台 -> 加密资产或 RWA | 已熟悉加密生态 | 平台、政策、链上来源解释 |

## 长期留存

如果美股是长期资产配置的一部分，未必需要频繁出金。但长期不动也不是“什么都不用管”。

需要提前准备：

- 账户清单：券商、银行、邮箱、手机号、双因素工具。
- 家人知情方案：至少让可信任的人知道资产在哪里。
- 税务资料包：年度税表、月结单、交易记录、股息预扣记录。
- 继承安排：非美国税务居民持有美国资产可能涉及遗产税，具体规则以 IRS 和专业顾问意见为准。

## 回到香港银行

```mermaid
flowchart LR
  A["目标券商"] --> B["香港银行"]
  B --> C["境内银行或境内消费"]
```

这条路好理解，也最容易被普通银行接受。前提是资金来源、交易收益和税务处理都能说清楚。香港账户可能产生 CRS 年度摘要，所以大额回款前建议先整理好：

- 券商年度税表。
- 逐笔或年度交易明细。
- 原始入金记录。
- 已申报或待申报税务材料。

## 进入美国银行后消费

如果已经有美国银行账户，可以先把券商资金出到美国银行，再用于海外消费、学费、旅行或外卡支付。这种做法的核心价值是减少来回跨境结汇。

但要注意：

- 美国银行也会做反洗钱和资金来源审查。
- 国内平台绑定外卡消费仍可能产生支付记录、限额和手续费。
- 账户长期不用、地址失效、手机号失效都可能带来登录和风控问题。

## Wise 小额出口

Wise 的优势是灵活，适合小额、清晰、低频的资金流。它不是长期大额仓库。

使用时建议：

1. 只做本人同名账户之间的转账。
2. 避免短期多笔拆分或多人收付款。
3. 提前准备券商出金记录和资金来源说明。
4. 账户被问询时，第一时间提供完整材料，不要临时编理由。

## 不建议

- 没准备税务资料就把大额资金直接回到境内。
- 通过他人账户“过一下”，但没有真实关系和凭证。
- 把清晰的券商资金转成自己并不熟悉的加密资产。
- 只比较手续费，不考虑冻结后能否申诉。

## 出金前资料包

| 材料 | 用途 |
| --- | --- |
| 券商月结单/年度税表 | 证明资金来自证券账户 |
| 交易明细 | 计算收益和解释盈亏 |
| 入金回单 | 证明本金来源 |
| 收款银行资料 | 确认同名、币种、账号和地址 |
| 税务处理记录 | 回答是否申报、如何申报 |
| 备用联系方式 | 处理银行或券商问询 |
